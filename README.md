<h1>Snake Game</h1>
A simple snake game made from scratch using C++ and Raylib as main library for rendering.
<br><br><br>
<strong>Releases</strong><br>
You can find binary files for both linux and Windows OS in the release section if you dont want to compile the code, else, please keep reading
<br><br><br>
<strong>How To Build The Code</strong><br>
You need as requirement to install Raylib on your machine beside a c++ compiler (in this project GNU compiler was used).
THen you will find the Makefile file that contains the base command to compile the files into an executable one (depending on the OS)
